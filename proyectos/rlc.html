¿De que trata?

RLC o Room Light Changer es un dispositivo iot que permite controlar la luz de una habitación de manera remota o llevando un control del número de personas dentro del cuarto y apagándola automáticamente cuando se encuentre vacío.


El dispositivo es una evolución de un proyecto previo Room Occupancy Counter el cual tenía como propósito limitar el número máximo de personas en una habitación con la llegada de la pandemia. Usando este mismo concepto mantuve el funcionamiento original pero cambié el cerebro del sistema de usar un arduino nano a usar un esp32 y darle cualidades de conexión wifi lo que me permitió crear una aplicación en mi teléfono para encender o apagar la luz de mi habitación con solo 1 click.


Este dispositivo contaba con muchas otras carácterísticas además de las ya mencionadas, entre ellas están:

  • Medidor de humedad.
  • Medidor de temperatura.
  • Interfaz de control local con botones.
  • Pantalla LCD para la visualización de datos de forma local.
  • Alarma para indicar cuando hay muchas personas.
  • Conexión wifi.
  • Conexión bluetooth.
