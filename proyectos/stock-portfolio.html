¿De que trata?

Mi nuevo proyecto consiste en una página web diseñada para visualizar diariamente el valor neto de mis inversiones. Esta plataforma se ejecuta en un servidor web alojado en una Raspberry Pi 4, brindándome una solución accesible y autónoma para monitorear mis inversiones en cualquier momento. Con esta página web, puedo obtener una visión clara y actualizada de mi cartera de inversiones en tiempo real. Utilizando la potencia de la Raspberry Pi 4, los datos financieros se recopilan, procesan y se presentan en una interfaz intuitiva y fácil de usar.


Gracias a la naturaleza autónoma de la Raspberry Pi 4, puedo acceder a la página web desde cualquier dispositivo haciendo uso de un servicio para poner en una ip pública lo que redirige mi Raspberry por medio de ssh, lo que me brinda flexibilidad y comodidad para realizar un seguimiento continuo de mis inversiones, incluso cuando estoy fuera de casa.
